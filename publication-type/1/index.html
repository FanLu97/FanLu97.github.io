Urban Radiance Field Representation with Deformable Neural Mesh Primitives
This paper proposes a novel neural rendering framework for urban scenes. We represent the urban scene as a set of deformable neural mesh primitives (DNMPs), which are able to capture the complex geometry and appearance of urban scenes.
Fan Lu, Yan Xu, Guang Chen, Hongsheng Li, Kwan-Yee Lin, Changjun Jiang

HRegNet: A Hierarchical Network for Large-scale Outdoor LiDAR Point Cloud Registration
This paper proposes a hierarchical network named HRegNet, which leverages rich features in deeper layer and precise location information of keypoints in shallower layers for robust and accurate LiDAR point cloud registration.
Fan Lu, Guang Chen, Yinlong Liu, Lijun Zhang, Sanqing Qu, Shu Liu, Rongqi Gu, Changjun Jiang

PointINet: Point Cloud Frame Interpolation Network
This paper propose a novel framework, namely Point Cloud Frame Interpolation Network (PointINet). Based on the proposed method, the low frame rate point cloud streams can be upsampled to higher frame rates.
Fan Lu, Guang Chen, Sanqing Qu, Zhijun Li, Yinlong Liu, Alois Knoll

RSKDD-Net: Random Sample-based Keypoint Detector and Descriptor
This paper proposes Random Sample-based Keypoint Detector and Descriptor Network (RSKDD-Net) for large scale point cloud registration. The key idea is using random sampling to efficiently select candidate points and generate keypoints and descriptors.
Fan Lu, Guang Chen, Yinlong Liu, Zhongnan Qu, Alois Knoll
